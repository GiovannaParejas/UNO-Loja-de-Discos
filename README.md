# UNO — Loja de CDs e Discos 🎵

Site institucional da **UNO**, uma loja especializada em CDs, discos de vinil, raridades e importados. Projeto desenvolvido para o Checkpoint 2 da disciplina de Frontend — FIAP.

---

## 🔗 Link do projeto

[Acesse aqui](https://giovannaparejas.github.io/UNO-Loja-de-Discos/)

---

## 📄 Páginas

| Página | Descrição |
|---|---|
| `index.html` | Home com loading screen, hero, gêneros, destaques e CTA |
| `portfolio.html` | Catálogo completo com filtros dinâmicos |
| `sobre.html` | História da loja, linha do tempo e equipe |
| `contato.html` | Formulário de contato e informações |

---

## ⚙️ Funcionalidades

- ✅ Loading screen com disco de vinil girando e barra de progresso
- ✅ Menu responsivo com hamburguer no mobile
- ✅ Filtros dinâmicos no portfólio por tipo, gênero e badge
- ✅ Navegação por gênero com redirecionamento para o portfólio filtrado
- ✅ Formulário de contato com validação em JavaScript
- ✅ Layout responsivo para desktop, tablet e mobile
- ✅ Linha do tempo interativa na página Sobre
- ✅ Hospedado no GitHub Pages
